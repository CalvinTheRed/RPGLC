# IncludesDamageType

```c#
{
  "condition": "includes_damage_type",
  "damage_type": <string>
}
```

Returns true if the subevent's damage type matches the condition's subevent.

###### Applicable Subevents
- `CriticalHitDamageCollection`
- `DamageAffinity`
- `DamageCollection`
- `DamageDelivery`
- `DamageRoll`
